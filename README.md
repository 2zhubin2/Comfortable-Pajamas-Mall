# Comfortable-Pajamas-Mall
“Comfortable Pajamas Mall”technical support
"Comfortable pajamas mall" is an app selling pajamas. Its main function is to sell pajamas. In this app, a large number of pajamas are provided and classified according to the categories of men's pajamas, women's pajamas, children's pajamas, lovers' pajamas, etc. Users can click the product to place an order directly, or add the product to the shopping cart for batch operation. And we also provide a circle communication function, users can display their own products, or make complaints about their problems. We will also push some articles about Pajama cleaning and Pajama care for users to read.
